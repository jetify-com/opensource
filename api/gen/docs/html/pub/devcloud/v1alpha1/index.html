SandboxState

SandboxState represents the state of a sandbox and maps to workstationspb.Workstation_State

in the GCP Workstations API.

NameNumberDescription
SANDBOX_STATE_UNSPECIFIED 0

Do not use.
SANDBOX_STATE_STARTING 1

The workstation is not yet ready to accept requests from users but will be soon.
SANDBOX_STATE_RUNNING 2

The workstation is ready to accept requests from users.
SANDBOX_STATE_STOPPING 3

The workstation is being stopped.
SANDBOX_STATE_STOPPED 4

The workstation is stopped and will not be able to receive requests until it is started.
